# Change Log

All notable changes to the "Kaimandres" extension will be documented in this file.

## [1.0.0] - 2025-12-25

### Initial Release
- Official release of Kaimandres for VS Code as the official port of my [kaimandres.nvim](https://github.com/MartelleV/kaimandres.nvim) theme; maintaining the original palette and highlighting logics.
- Complete Workbench theming (Editor, Sidebar, Activity Bar, Terminal, Popups).
- Support for italic keywords and bold control flow statements.
